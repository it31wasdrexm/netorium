# Netorium CLI

Netorium is a cross-platform command-line tool for building-level network access control.

Use it to prepare configuration, organize devices by zones, inspect local health, and
build toward safe firewall, monitoring, directory, reporting, and alert workflows.

## Start Here

```bash
netorium --help
netorium version
netorium doctor
netorium config init
netorium config validate
netorium update show
netorium-agent --help
```
